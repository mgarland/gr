## Usage

Load file-name, sort based on the sort-option and display on screen

    $ java -jar gr-0.1.0-standalone.jar file-name sort-option

## Options

Sort option: 1, 2 or 3

## Examples

java -jar gr-0.1.0-standalone.jar "/tmp/input.txt" 1
